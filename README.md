# Hayden Web3 - Welcome Site

A fast, minimal React website with 8-bit styled text and fade-in animation.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

## Build

Create a production build:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Features

- Fast response time with Vite
- White background with centered content
- 8-bit pixelated font (Press Start 2P)
- Smooth fade-in animation
- Responsive design
